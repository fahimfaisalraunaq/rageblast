# RageBlast 🔥

Anonymous one-shot rage exchange platform.

RageBlast allows users to send an anonymous emotional message and receive exactly one anonymous reply. After the response, the thread automatically closes and both users disappear from each other's experience.

## Features

* Anonymous interactions
* One-shot messaging
* AI personality responders
* Randomized response styles
* Automatic thread closure
* No usernames or social graph
* Future personality stats and ratings system

## Current MVP

* Send anonymous message
* Simulated AI personalities:

  * Chaos Goblin
  * Wise Monk
  * Savage Roaster
  * Meme Brain
  * Calm Therapist
* Instant response generation
